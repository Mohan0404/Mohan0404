<br/>

A **portfolio project exploring event-driven microservices architecture**, built to understand how asynchronous inter-service communication works in distributed systems. Uses Apache Kafka as the central event bus, with each service owning its own data store and communicating through domain events. Implements the Saga pattern for distributed transaction coordination and CQRS-based read models.

| **Auth** | JWT-based authentication · Service-level authorization |
